# Record

Finalized documents that passed human review in `01_work`. Do not edit after filing.

File naming: `yyyy-MM-dd-[a-z0-9-].md` (e.g. `2026-04-03-auth-adr.md`)

Version suffix is removed when filing (e.g. `auth-design-v3.md` -> `auth-design.md`).

Categories (use prefix in filename):
- **research** -- Reports from research and analysis.
- **decision** -- Product or architecture decision records (ADR).
- **compliance** -- Privacy, legal, and regulatory documents (e.g. Privacy by Design, data flow descriptions, vendor data processing records).

## Ownership

- **Writer**: Claude moves files here upon human approval. Version suffix is stripped.
- **Editor**: No one. Filed documents are immutable.
- **Reader**: Anyone.
