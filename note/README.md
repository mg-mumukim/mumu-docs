# Note

Source material, references, and project artifacts.

File naming: `yyyy-MM-dd-[a-z0-9-].md` (e.g. `2026-04-03-auth-spec.md`)

## Subdirectories

| Directory | Writer | Description |
|-----------|--------|-------------|
| `source/` | Human only | Raw inputs: meeting notes, system context, plans, prompts, CSVs, discussion notes. |
| `task/` | Human or Claude | Requests (`-request`) for `/write-draft`, plans (`-draft-plan`, `-report-plan`) for `/write-draft` and `/write-report`. |
