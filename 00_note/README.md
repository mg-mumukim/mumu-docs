# Note

Raw inputs. Files here serve as source material for documents in `01_work`.

- thoughts
- templates
- specifications
- meeting notes
- slack threads

File naming: `yyyy-MM-dd-[a-z0-9-].md` (e.g. `2026-04-03-auth-spec.md`)

## Subdirectories

- `context/` -- Persistent reference material (e.g. profile, team info). Git-tracked.
- `idea/` -- Quick ideas captured via `/idea` skill. Git-tracked.
- `personal/` -- Private notes (1:1, evaluations). Git-ignored.
- `offical-docs/` -- External reference documents.

## Ownership

- **Writer**: Human only. Claude does not create or modify files here.
- **Reader**: Claude reads these files as source material when drafting documents in `01_work`.
